# /status-report — Generate Project Status Report

Generate a comprehensive status report for the CollabSpace v2 project.

## Instructions
1. Read `tasks.md` for task status.
2. Read `release-notes.md` for completed work.
3. Read `requirements-v2.md` to calculate requirement coverage.
4. Generate a report with:
   - **Overall Progress:** % of tasks done per milestone
   - **Current Sprint:** What's in progress, who's working on what
   - **Completed:** Recent completions
   - **Blocked/At Risk:** Any tasks that are stuck
   - **Next Up:** Suggested next tasks to assign
   - **Requirement Coverage:** Which FRs and NFRs are implemented

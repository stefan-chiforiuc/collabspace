# /git-status — Show Git Flow Status

Display current branch, all agent branches, working tree status, and recent commits.

## Instructions

1. Run:
```bash
bash .claude/memory-db/git-flow-helper.sh status
```

2. Also show agent branches:
```bash
bash .claude/memory-db/git-flow-helper.sh list-branches
```

3. Summarize: which agents are working on what branches, any uncommitted changes.

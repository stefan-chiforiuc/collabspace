# CollabSpace v2 — Project Memory Index

## Primary Memory: Vector DB
All project memories are stored in the **local vector database** at `.claude/memory-db/`.
Use `node .claude/memory-db/memory-store.mjs` to add, search, list, update, and export memories.
See `CLAUDE.md` for full usage instructions.

  Additional Collaboration Ideas

  - Shared Whiteboard — draw, sketch, annotate together in real-time
  - Shared Timer/Stopwatch — for timeboxing discussions or activities
  - Collaborative Notepad — simple shared text editing (meeting notes)
  - Retro Board — columns for "went well / didn't go well / action items" with sticky notes
  - Dot Voting — everyone gets N dots to distribute across options (more flexible than polls)
  - Q&A Queue — raise hand / submit questions, host can manage order
  - Reaction Bar — quick emoji reactions visible to all (thumbs up, confused, etc.)
  - Shared Link Collection — anyone can drop links, all see them organized
  - Decision Matrix — criteria vs. options grid, everyone scores, weighted results shown
